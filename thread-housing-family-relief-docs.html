Relief Moderator

Checklist starter

Feb 15, 2026

Separate documents by claim type before uploading or filing. Mixed files are one of the most common reasons community members report delays.

  • Rent claims: contract plus payment trail.
  • Mortgage interest: lender statement by year.
  • Dependents: identity and support evidence.

Expert note

Feb 13, 2026

Label every file with taxpayer name, claim type, and period. Clear naming conventions save significant time during verification and amendments.

Claim Pack Structure

  • Create one folder per claim type.
  • Add an index page for each folder.
  • Match totals to your return fields.
  • Store source scans and originals safely.
